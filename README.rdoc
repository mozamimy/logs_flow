# LogsFlow

[CloudWatch Logs](https://docs.aws.amazon.com/AmazonCloudWatch/latest/DeveloperGuide/WhatIsCloudWatchLogs.html) event viewer.

## Installation

```sh
bundle install
vi config/initializers/logs_flow.rb
bundle exec rails s
```

Then go to `http://localhost:3000`.
